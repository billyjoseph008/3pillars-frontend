# 3 Pillars Technical Assessment

This repository contains my implementation attempt for the 3 Pillars technical assessment.

The exercise consisted of two parts:

1. **Middleware Service**
   - Create a middleware that generates random numeric "steps".
   - The system should expose a mapping of keys **1–10** to randomly generated numeric values.

2. **Full Stack Feature**
   - Build a full application starting from the database layer to the frontend.
   - The backend exposes users from the following schema:

```sql
create table random_user(
  id SERIAL AUTO_INCREMENT PRIMARY KEY,
  firstname VARCHAR(50),
  lastname VARCHAR(50),
  age INT NOT NULL,
  country VARCHAR(100),
  avatar VARCHAR(250),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

The frontend should display these users in a **grid view with avatars**, include a **filter input**, and show the **random step values generated by the middleware**.

## Stack Used

* **Frontend:** React (Hooks)
* **Backend:** Java Spring Boot + JPA
* **Database:** MySQL

Due to time constraints the full implementation was not completed, but the repository demonstrates the intended architecture and core integration between frontend, backend, and database.
